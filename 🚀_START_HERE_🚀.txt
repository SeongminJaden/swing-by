              🆘 **문제 해결 (문제 시 여기서 찾으세요)**


Docker 관련:
  • "Docker 없음" → Docker Desktop 설치
  • 포트 충돌 → docker-compose.yml 포트 변경
  • 메모리 부족 → Docker 메모리 8GB+ 할당

Rust 관련:
  • "Cargo 없음" → Rust 설치
  • 컴파일 오류 → cargo clean && cargo build

Git 관련:
  • "Git 없음" → Git 설치
  • 커밋 실패 → init_git.bat 다시 실행

Ollama 관련:
  • "모델 없음" → quick-start.bat에서 자동 다운로드
  • API 연결 실패 → docker logs -f ai_agent_ollama
