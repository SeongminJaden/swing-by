                         💻 핵심 명령어
───────────────────────────────────────────────────────────────────────────────

개발
  cargo run                   # 실행
  cargo test                  # 테스트
  cargo fmt                   # 포맷
  cargo clippy                # 린트
  RUST_LOG=debug cargo run   # 상세 로그

디버깅
  F5                         # VSCode 디버거
  RUST_BACKTRACE=1 cargo run # 백트레이스

배포
  ./deploy.bat               # 배포 패키지 생성

Docker
  docker-compose up -d       # Ollama 시작
  docker logs -f ai_agent_ollama  # 로그 확인
  docker exec ai_agent_ollama ollama list  # 모델 확인

Docker 없음
